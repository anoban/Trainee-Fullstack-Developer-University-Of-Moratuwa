JavaScript is an object oriented programming language! (OOP).
Objects in JS have properties & behaviours. 

 Here's how JavaScript syntax looks;
            let object = {
                colour: #FF87R;
                border: 20px;
                border-color: #FD34F2;
                width: 100%;
                enlarge: function() {    }
                rotate: function() {    }
            };

In JavaScript behaviours are defined as functions and properties are defined as key: values pairs!
A property of a JS object can be accessed by calling object.property
And behaviours of a JS object can be accessed by calling object.behaviour()

Classes are critical in OOP!. Rewriting the same code with minor changes for similar JS objects will result in redundant code! Classes help reduce code redundancy and improves code reuse! 

            class myclass {
                constructor(value) {
                this.property1 = value;
                this.property2 = 12px;
                this.property3 = bold;
                this.behaviour = behaviour: myfunction(){    };
                }
            }; 


            Below we minimize code redundancy by creating new JS objects using classes!.
            Here code is minimal since we pass all the common object attributes as default class 
            attributes and specify only the attributes that are unique to objects manually!

            let myclass_object_1 = new myclass("#56F#FD");
            let myclass_object_2 = new myclass("#DF34F4");
