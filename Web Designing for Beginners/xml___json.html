XML

XML is an old markup language. XML stands for eXtensible Markup Language.
It is much similar to HTML.
XML was designed to store and transport data and is self descriptive.
XML is commonly used in data distribution over the internet.
In XML there aren't any predefined tags as in HTML. Users can create any tags as they please!
XML simplifies data availability, sharing, transport and platform changes.

            <?xml version="1.0" encoding="UTF-8"?>
            <note>
                <to> Janice </to>
                <from> Wattmann </from>
                <heading> Reminder </heading>
                <body> Don't forget me this weekend! </body>
                <sender> Jeanne Osbert <sender>
                <greet> Wishing you well <greet>
            </note>

JSON

JSON stands for JavaScript Object Notation.
It is a text file format for storing and transporting data.
It is a relatively new entry.
JSON is a lightweight data exchange format that is used to exchange information between computers.
It is language independent.
