JavaScript provides 5 different types of loops.

for loop
iterates through a block of code for a given number of times.
for/in loop
loops through the properties of an object
for/of loop
loops through the values of an inerrable JS object
while loop
loops through a block of code while the given condition is true
do/while loop
loops through a block of code while the given condition is true. However the loop body expressions will be executed in the first iteration whether the loop condition is met or not.

It is important to use the let keyword to declare local variables inside JavaScript loops, since declarations using var keyword will make those variables global. 

            Syntax of JS for loop is quite similar to C++ for loops!
            let text = "";       /* declaration & instantiation */
            for (let i = 0; i < 10; i++) {
                text += "The number is " + i + "<br>";
            }


            Syntax of JS for/in loop in quite similar to Python for loops! 
            Hovever they are not similar functionally. Here this loop iterates through the keys of JS objects!.
            const Natalie = {fname:"Natalie", lname:"Martinez", age:27, race:"Hispanic", married:0}; 
            let text = "";
            for (let key in Natalie) {
                text += Natalie[key];
            }

            "key in Natalie" loops over the keys of the JS object Natalie, and these keys are used to 
            extract their cognate values by object[key] method!. 
            Then the extracted values are concatenated with the string text.


            Syntax of JS for/of loop! This is functionally similar to Python's for loops but not so syntactically!
            let chararray = ['A', 'N', 'y', 'q', 'D', 'L'];         /* a JS array */
            let text = "";
            for (let i of chararray) {
                text += i;
            }


            While loops in JS loops over the loop body as long as the loop condition is true!
            Syntax:
            let i = 0;
            let sum = 0;
            while (i < 100) {
                sum += i;
                i++;
            }


            Do while loop in JS is identical to C++ do while loops. Loop body will execute regardless of
            the loop condition once but subsequent iterations will only happen if the loop condition is
            met. 
            Syntax:
            let i = 100;
            sum = 0;
            do {
                sum += i;    
            } while (i < 10)
