Every HTML element can be considered as a box!
This box model comprises of boxes enclosed in boxes!
The outermost box is the margin!
The layer just inwards margin is border!
And the layer right next to the HTML element is padding!
Thus when going from the edge of an HTML element, there are padding, border and margin!

Margins of adjacent HTML objects can overlap! This will result in the effective margin being that of the object having highest margin value!
