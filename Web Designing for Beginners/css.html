Let's get started with CSS

CSS styling can be applied to HTML documents to enhance the visual appeal of the webpage and improve user experience! CSS styling can be applied externally/internally/inline! This document is styled with an external CSS document! When using external CSS files to style webpages, we can modify the entire outlook of a webpage by modifying just one page!


CSS syntax is essentially key:values pairs! Initially one has to declare which HTML tag the styles defined downstream apply to and then property and the styles are defined inside curly braces!


selector { declaration : value;
           declaration: value;
           declaration: value; }


HTML_tag { property1 : value;
           property2: value;
           property3: value; }


.user_defined_class { property1 : value;
           property2: value;
           property3: value; }


When we want to apply styling to just a few HTML elements of one type but not for all in the entire document, classes can be made for HTML tags, which can then be used to apply CSS to just that subset of HTML tag element!

This line is styled with an internal CSS code chunk! i.e. with the following CSS code is in the same HTML document! 

  .csssyntax, pre { font-size: 22px;
                                color: navy;
                                font-family: Gill sans;
                                font-weight: 400; }

This line is styled with an inline css! Inline CSS styling has the highest precedence!, followed by internal, external CSS styles with browser defaults at last! The inline css applied to this paragraph is <p style="font-family:Consolas; color:deeppink; font-size:24px">
