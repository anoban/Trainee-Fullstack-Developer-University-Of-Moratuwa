JavaScript events are fundamental to facilitate user interactivity in a webpage! Events are triggers that return a premediated resonse back to user! HTML documents have many elelments where users provide inputs e.g. forms, buttons, checkboxes, input text boxes etc..
Click events happen when users interact with radio buttons, checkboxes, menu buttons etc..
Load event happens when the browser finishes loading a webpage!


Thus the web engine signals the programme when an event occurs.
This signal triggers a predesignated JavaScript code that brings out the desired response!
