AJAX

AJAX stands for Asynchronous JavaScript And XML.
It is not a programming language.
AJAX uses XML-HTTP request objects built into web browsers and JavaScript with HTML DOM.
It is the XML-HTTP request objects that communicate with servers.
AJAX allows users to make requests to servers without preloading the page.

Web browsers communicate with servers in following way:
An event occurs → Browser creates an XML-HTTP request object → Browser sends it over internet as HTTP request → Server receives & processes the HTTP request → Server creates a response and sends it over internet to the web browser → Browser receives & processes the data using JavaScript → Updates the page contents accordingly


REST APIs

REST stands for REpresentational State Transfer.
API stands for Application Programming Interface.
REST is a web standard based architecture that uses the HTTP protocol.
get method provides read-only access to a resource.
post method is used to create new resources.
delete method is used to remove an existing resource.
put method is used to update an existing resource or create a new resource.
